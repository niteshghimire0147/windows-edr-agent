## Purpose of This Tool

This tool was developed **solely for educational and defensive security purposes**. It is intended to help students, security researchers, system administrators, and blue team practitioners understand:

- How Windows processes and services behave at a system level
- How malware abuses parent-child process relationships for execution
- How persistence mechanisms are established through Windows services
- How to implement rule-based and behavior-based threat detection
- How to apply the MITRE ATT&CK framework in a practical defensive context
